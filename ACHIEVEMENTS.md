# 成就系统说明

## 新增成就

### 1. 参数调优大师 🔬
- **解锁条件**：在教学关卡的隐藏关卡中达到最高正确率（95%）
- **类型**：普通成就
- **前置条件**：教学关卡的节点必须连接正确
- **解锁方式**：
  1. 确保教学关卡的分类器节点连接正确：
     - classifier-out1 → target-in（红色方块到有目标）
     - classifier-out2 → no-target-in（蓝色方块到无目标）
  2. 点击教学关卡左下角的隐藏关卡按钮（🔬）
  3. 调整训练参数（学习率、批量大小、训练轮数、优化器）
  4. 点击"测试训练"查看准确率
  5. 当准确率达到 95% 时自动解锁

**重要提示**：
- 如果教学关卡连接错误，隐藏关卡中的准确率会反转（95%变成5%）
- 只有在连接正确的情况下才能解锁成就
- 连接错误时会显示"⚠️ 连接错误"警告

**最优参数组合（可达到95%准确率）**：
- 学习率：0.1 (+4%)
- 批量大小：32 (+4%)
- 训练轮数：20 (+4%)
- 优化器：Adam (+8%)
- 基础准确率：75%
- 最终准确率：75% + 20% = 95%

### 2. 反向工程师 🔄
- **解锁条件**：在教学关卡连接错误时达到极低准确率（≤10%）
- **类型**：隐藏成就
- **解锁方式**：
  1. 在教学关卡中故意将分类器节点连接错误：
     - classifier-out1 → no-target-in（红色方块到无目标）错误连接
     - classifier-out2 → target-in（蓝色方块到有目标）错误连接
  2. 点击"测试"按钮运行测试
  3. 当最终准确率 ≤ 10% 时自动解锁
- **特点**：
  - 这是一个"反向"成就，需要故意做错才能解锁
  - 连接错误时，准确率会反转（正确率变成错误率）
  - 未解锁时显示为 "???"
  - 解锁后显示"隐藏"徽章

**如何达到最低准确率**：
- 确保分类器连接完全错误（两个输出都接错）
- 基础准确率75%会反转为25%错误率
- 如果隐藏关卡参数调得很差，可以进一步降低准确率到10%以下

### 3. 隐藏探索者 🕵️
- **解锁条件**：完成所有隐藏关卡（3/3）
- **类型**：隐藏成就
- **解锁方式**：
  1. 完成教学关卡的隐藏关卡（点击保存并应用）
  2. 完成第二关的隐藏关卡（待开发）
  3. 完成第三关的隐藏关卡（待开发）
- **特点**：
  - 未解锁时显示为 "???"
  - 描述显示为 "完成特定条件解锁"
  - 图标模糊显示
  - 解锁后显示"隐藏"徽章

## 成就系统特性

### 成就解锁提示
- 解锁成就时会在屏幕右上角显示动画提示
- 紫色渐变背景，带发光效果
- 图标旋转动画
- 5秒后自动消失

### 成就显示
- 在个人中心查看所有成就
- 已解锁成就：蓝色边框高亮
- 未解锁成就：半透明显示
- 隐藏成就：紫色背景，图标模糊

## 开发调试

在浏览器控制台中可以使用以下命令：

```javascript
// 查看当前成就状态
achievementDebug.checkStatus()

// 解锁参数调优大师
achievementDebug.unlockParamMaster()

// 解锁反向工程师
achievementDebug.unlockReverseEngineer()

// 解锁隐藏探索者
achievementDebug.unlockHiddenExplorer()

// 重置所有成就
achievementDebug.resetAllAchievements()

// 模拟完成隐藏关卡
achievementDebug.completeHiddenLevel(1) // 完成第1个隐藏关卡
achievementDebug.completeHiddenLevel(2) // 完成第2个隐藏关卡
achievementDebug.completeHiddenLevel(3) // 完成第3个隐藏关卡
```

## 数据存储

成就数据存储在 localStorage 中：

- `achievement_param_master`: 参数调优大师成就
- `achievement_expert_tuner`: 专家系统调优师成就
- `achievement_reverse_engineer`: 反向工程师成就
- `achievement_hidden_explorer`: 隐藏探索者成就
- `hidden_levels_completed`: 已完成的隐藏关卡数量
- `tutorial_hidden_done`: 教学关卡隐藏关卡完成标记
