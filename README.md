# AI指令师：从零开始训练你的第一个AI

> 一款面向AI小白的机器学习教学游戏 —— 通过拖拽节点、搭建流水线，轻松理解过拟合、决策树、神经网络等核心概念。

## 📖 项目简介

本项目是一款基于 **Electron + React + Vite** 开发的桌面小游戏。玩家扮演“AI训练师”，通过可视化节点拖拽的方式，完成一系列机器学习挑战。游戏内置渐进式关卡、知识点小卡片、商店和个人中心，让零基础的玩家也能在趣味中掌握AI的基本思想。

## ✨ 核心功能

- **📚 关卡系统**：8个由浅入深的教学关卡，涵盖训练/预测、欠拟合、过拟合、决策树、随机森林、神经网络、特征工程、超参数调优。
- **🧩 可视化编程**：基于 React Flow 的节点编辑器，拖拽连接即可搭建机器学习流水线。
- **📊 实时训练反馈**：点击“训练”后展示准确率、损失曲线、混淆矩阵，并提供诊断建议。
- **🎴 知识点小卡片**：每关开始时弹出图文并茂的概念卡片，帮助玩家快速理解新知识。
- **🏪 商店系统**：使用通关获得的积分购买新节点、皮肤或提示道具。
- **👤 个人中心**：记录通关进度、最高评分、成就徽章，支持重置进度。
- **💡 智能诊断助手**：训练失败时给出具体错误原因（如“模型过拟合，试试减小树的深度”）并高亮问题节点。

## 🎮 关卡设计（8关）

| 关卡 | 核心概念     | 学习目标                                     |
| ---- | ------------ | -------------------------------------------- |
| 01   | 训练与预测   | 理解模型从数据中学习并应用到新数据的能力     |
| 02   | 欠拟合       | 识别模型过于简单导致的训练与测试准确率双低   |
| 03   | 过拟合       | 识别模型复杂度过高导致的训练好但测试差       |
| 04   | 决策树       | 掌握树模型的直观分类/回归逻辑                |
| 05   | 随机森林     | 理解集成学习如何通过多棵树投票提升效果       |
| 06   | 神经网络     | 了解多层神经元的信息传递与激活               |
| 07   | 特征工程     | 学习如何构造/筛选更有意义的特征              |
| 08   | 超参数调优   | 体验调整学习率、树深度等参数对模型的影响     |

每关均包含：
- 教学卡片（文字+示意图）
- 明确的通关目标（如准确率 > 90%）
- 可用的节点库（2~3种正确解法）
- 训练反馈 + 诊断提示

## 🗓️ 三周开发计划

### 第一周：搭建骨架与核心玩法

- **第1-3天**  
  - 使用 `create-electron-vite` 初始化项目  
  - 配置 React + TypeScript + Vite 开发环境  
  - 学习并集成 React Flow 库，搭建基础节点编辑器  
  - 实现游戏主窗口布局（关卡选择、节点面板、训练区）

- **第4-7天**  
  - 设计并实现前3个关卡（训练预测、欠拟合、过拟合）  
  - 实现简单的模拟训练逻辑（不依赖真实 ML 框架）  
  - 开发数据显示组件（准确率、损失曲线）  
  - 实现基础诊断提示（弹窗 + 文本建议）

### 第二周：丰富关卡与可视化体验

- **第8-11天**  
  - 实现关卡4-6（决策树、随机森林、神经网络）  
  - 完善可视化组件：训练曲线、混淆矩阵、特征重要性图  
  - 增加知识点小卡片弹窗系统（按关卡触发）

- **第12-14天**  
  - 实现关卡7-8（特征工程、超参数调优）  
  - 打磨小白引导：动态高亮、首次操作指引  
  - 开发商店系统（积分兑换节点/皮肤）  
  - 搭建个人中心页面（进度、成就、重置）

### 第三周：测试、优化与交付

- **第15-18天**  
  - 邀请2~3名非计算机专业的同学进行用户测试  
  - 根据反馈调整关卡难度和诊断提示的准确性  
  - 优化 React Flow 性能（节点较多时的渲染与拖拽流畅度）  
  - 修复已知 bug（训练逻辑、状态同步等）

- **第19-21天**  
  - 使用 `electron-builder` 打包 Windows / macOS 应用  
  - 录制2~3分钟的演示视频（从关卡1到关卡8的流畅演示）  
  - 撰写课程项目报告（设计理念、技术实现、测试反馈）  
  - 准备答辩 PPT，整理项目文档与源码

## 🚀 快速开始

### 环境要求
- Node.js 18+ 
- npm / yarn / pnpm

### 安装与运行

```bash
# 克隆项目
git clone https://github.com/your-username/ai-commander-game.git
cd ai-commander-game

# 安装依赖
npm install

# 启动开发环境（自动打开 Electron 窗口）
npm run dev

# 打包应用
npm run build:win   # Windows
npm run build:mac   # macOS
npm run build:linux # Linux
```
