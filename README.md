# AI训练师 - 机器学习可视化教学游戏

> 一款基于 Electron + React + TypeScript 开发的机器学习教学游戏，通过可视化节点连接和数据流动画，让玩家在游戏中理解AI的核心概念。

## 📖 项目简介

**AI训练师**是一款面向AI初学者的桌面游戏，玩家通过拖拽节点、连接数据流、调整参数等方式，完成一系列机器学习挑战。游戏采用渐进式关卡设计，从基础的数据分类到复杂的负载均衡和决策树，让零基础的玩家也能在趣味中掌握AI的基本思想。

## ✨ 核心功能

### 🎮 游戏系统
- **用户认证系统**：基于 Supabase 的登录/注册功能，支持用户数据持久化
- **关卡选择界面**：4个主线关卡 + 3个隐藏关卡，渐进式难度设计
- **商店系统**：使用通关获得的金币购买道具和装备
- **个人中心**：查看学习进度、成就徽章、积分统计
- **实时金币系统**：完成关卡获得金币奖励，支持跨页面实时更新

### 🧩 关卡系统

#### 主线关卡（4关）

| 关卡 | 名称 | 核心概念 | 学习目标 | 通关奖励 |
|------|------|----------|----------|----------|
| 第一关 | 教学关卡 | 数据分类 | 理解分类器如何将不同颜色的数据分流到不同输出 | 100金币 |
| 第二关 | 专家系统 | 数据清洗 | 学习如何使用专家系统过滤和分类数据 | 200金币 |
| 第三关 | 负载均衡 | 数据分发 | 掌握负载均衡器如何将数据均匀分配到多个目标 | 150金币 |
| 第四关 | 决策树 | 路由决策 | 理解决策树如何根据条件进行数据路由 | 300金币 |

#### 隐藏关卡（3关）

| 关卡 | 触发条件 | 挑战内容 | 成就奖励 |
|------|----------|----------|----------|
| 第一关隐藏 | 主关卡通关 | 参数调优：达到95%以上正确率 | 参数调优大师 |
| 第二关隐藏 | 主关卡通关 | 专家系统调优：达到100%正确率 | 专家系统调优师 |
| 第四关隐藏 | 主关卡通关 | 构建决策树路由器：准确率≥80%，负载差异≤30% | 隐藏探索者 |

### 🎯 游戏机制

#### 可视化节点系统
- **拖拽放置**：从节点库拖拽节点到画布
- **连接线绘制**：点击输出点拖动到输入点建立连接
- **实时数据流**：测试时显示彩色方块沿连接线流动
- **节点容量管理**：节点有最大容量限制，超载时数据会被阻塞

#### 测试与反馈系统
- **实时准确率显示**：柱状图显示当前准确率
- **进度条动画**：显示测试进度
- **通关判定**：达到目标准确率后显示通关弹窗
- **失败提示**：未达标时给出改进建议

#### 参数调优系统
- **隐藏参数面板**：调整置信度阈值、规则权重、过滤强度等参数
- **实时训练反馈**：显示当前参数组合的准确率
- **最优解探索**：寻找最佳参数组合达到100%准确率

### 🏆 成就系统

共8个成就徽章，分为显性和隐藏两类：

**显性成就**
- 🎓 **初出茅庐**：完成教学关卡
- 📊 **数据大师**：完成所有数据相关关卡
- 🤖 **模型专家**：完成所有模型关卡
- ⭐ **完美主义者**：所有关卡获得满分

**隐藏成就**
- 🎛️ **参数调优大师**：第一关隐藏关卡达到95%以上正确率
- 🔧 **专家系统调优师**：第二关隐藏关卡达到100%正确率
- 🔍 **隐藏探索者**：完成所有隐藏关卡（3/3）
- 🔄 **反向工程师**：在教学关卡连接错误时达到极低准确率（≤10%）

### 🎨 UI/UX 特性

- **响应式布局**：支持窗口缩放和最小化
- **缩放功能**：画布支持滚轮缩放（50%-200%）
- **动画效果**：
  - 数据流粒子动画
  - 成就解锁动画
  - 购买成功/失败弹窗动画
  - 奖励获得动画
- **教程引导**：每关首次进入显示动画教程
- **实时更新**：金币、进度、成就状态跨页面实时同步

## 🎯 关卡详细设计

### 第一关：教学关卡（数据分类）

**学习目标**：理解分类器如何将不同颜色的数据分流

**节点库**：
- 分类器（1个）：有1个输入、2个输出

**通关条件**：
- 连接正确（红色→目标，蓝色→非目标）
- 准确率 ≥ 75%

**隐藏关卡**：
- 调整参数达到95%以上正确率
- 解锁成就：参数调优大师

**特殊成就**：
- 反向工程师：故意连接错误，使正确率降至10%或以下

### 第二关：专家系统（数据清洗）

**学习目标**：学习如何使用专家系统过滤和分类数据

**节点库**：
- 专家系统（1个）：有1个输入、2个输出，可切换过滤模式（红/绿/蓝）
- 垃圾桶（1个）：只有输入，用于丢弃不需要的数据

**通关条件**：
- 正确率 ≥ 85%

**隐藏关卡**：
- 调整置信度阈值、规则权重、过滤强度三个参数
- 达到100%正确率
- 解锁成就：专家系统调优师

### 第三关：负载均衡（数据分发）

**学习目标**：掌握负载均衡器如何将数据均匀分配

**节点库**：
- 负载均衡器（最多3个）：有1个输入、2个输出，轮询分配数据

**通关条件**：
- 所有4个目标都收集到8个红色方块
- 准确率100%
- 时间限制：40秒

**特点**：
- 需要合理配置多个负载均衡器
- 实现数据的均匀分发

### 第四关：决策树（路由决策）

**学习目标**：理解决策树如何根据条件进行数据路由

**节点库**：
- 负载均衡器（最多10个）
- 分类器（最多4个）：处理延迟0.5秒
- 垃圾桶（1个）

**通关条件**：
- 收集100个方块
- 准确率 ≥ 75%（至少75个红色方块）
- 时间限制：180秒

**隐藏关卡**：
- 使用逻辑块构建决策树路由器
- 准确率 ≥ 80%
- 负载差异 ≤ 30%
- 解锁成就：隐藏探索者

## 🚀 快速开始

### 环境要求
- Node.js 18+
- npm / yarn / pnpm

### 安装与运行

```bash
# 克隆项目
git clone https://github.com/Wangaquatic/import-ai.git
cd import-ai

# 安装依赖
npm install

# 启动开发环境
npm run dev

# 打包应用
npm run build:win   # Windows
npm run build:mac   # macOS
npm run build:linux # Linux
```

### 数据库配置

项目使用 Supabase 作为后端数据库，需要配置以下环境变量：

1. 复制 `.env.example` 为 `.env`
2. 填入你的 Supabase 项目信息：
```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

3. 执行数据库初始化脚本（位于 `supabase/` 目录）
